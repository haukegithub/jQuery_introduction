Warum jQuery "einfach" ist
  • Klasse(n) hinzufügen:
    • Mit JavaScript:
      var c = document
      .getElementsByClassName("cl"); var i; for (i = 0; i < c.length; i++) { c[i].className =
      c[i].className + " selected"; }
  •  
    • Mit jQuery:
      $(".cl").addClass("selected");
Funktionen
  • Event handler, z.B.:
    • Mausklick abfangen: $("button").click(function() {
      // Zugriff mit $(this)
      });
    • Tasten abfangen: $("body").keydown(function(e) {
      // e incl. pressed key etc.
      }).keyup(function() {
      // On key release
      });
Funktionen
  • Elemente entfernen, z.B.:
    • Auswahl entfernen: $(".sub").last().remove();
    • Auswahl ersetzen: $(".sub")
      .replaceWith("<h2>Titel</h2>");
    • Auswahl ausblenden: $("h2").hide();
    • Auswahl einblenden: $("h2").show();
Funktionen
  • Elemente hinzufügen, z.B.:
    • In Auswahl als letztes hinzufügen: $("#cont div")
      .append("<b>Huhu</b>");
    • Nach Auswahl hinzufügen: $("#cont div")
      .after("<b>Huhu</b>");
    • Wrappen der Auswahl: $("#cont div b").wrap("<p>");
Funktionen
  • HTML, Text, Attribute, z.B.:
    • Inhalt mit HTML überschreiben: $("#_3").html("<i>!</i>");
    • Inhalt mit Text überschreiben: $("#_3").text("<i>!</i>");
    • Attribut setzen: $("#_2").attr("dies", "das");
    • Attribut abfragen: $("#_2").attr("dies");
Funktionen
  • CSS und Klassen, z.B.:
    • Style-Attribut: $("h1").css("margin", "5px");
    • Klasse hinzufügen: $("p.sub:first").addClass("wow");
    • Klasse entfernen: $(".wow").removeClass("sub");
    • Klasse abfragen: if ($("p").hasClass("wow")) {};
Funktionen
  • Navigation, z.B.:
    • Index: $("p:eq(1)");
    • Nächstes: $("p:eq(1)").next();
    • Voriges: $("p:eq(1)").prev();
    • Eltern: $("p:eq(1)").parent();
    • Kind(er): $("p:eq(1)").parent()
      .children();
    • Alle danach: $("p:eq(1)").nextAll();
    • Alle davor: $("p:eq(1)").prevAll();
Variablen und Funktionen
  • In Variable speichern:
    • let list = $("div, h1");
  • Funktionen anwenden:
    • Finden von Elementen in Auswahl:
      list.find("h1");
    • Über Auswahl iterieren:
      list.each(function() {
      let thisEl = $(this);
      });
Elemente selektieren
  • Weitere Möglichkeiten, z.B.:
    • Mehrere Elemente: $("div, h1");
    • Erstes Element: $("p:first");
    • Letztes Element: $("p:last");
    • Ungerade (Index): $("p:odd");
    • Inhalt: $("p:contains('TEST')");
    • Enthält: $("p:has('span')");
    • Negierung: $("p:not('#_2')");
Elemente selektieren
  • Nach CSS-Regeln, z.B.:
    • HTML-Element(e): $("div");
    • Element mit ID: $("p#_1");
    • ID: $("#_3");
    • Klasse(n): $(".sub");
    • Vorfahre / Nachfahre(n): $("#cont p");
    • Eltern / Kind(er): $("#cont > p");
    • Attribut(e) beginn(t/en) mit: $("a[href^=https]");
